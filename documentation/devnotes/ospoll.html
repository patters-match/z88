OS_Poll, poll for an application

RST 20H, DEFW $FC06

IN: 

     IX = 0, fetch first application in the machine
     IX <> 0 (current application handle), fetch next application
OUT, if call successfull: 
     Fc = 0
     IX = new application handle
OUT, if call failed: 
     Fc = 1
          A = RC_EOF, no more applications available (IX now invalid)
Registers changed after return: 
     ..BCDEHL/..IY same
     AF....../IX.. different
Notes:
Internal operating system usage.
An invalid input application handle also results in RC_EOF.
