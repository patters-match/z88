OS_Isq, initialize prefix sequence

RST 20H, DEFW $D006

IN: HL = sequence buffer

OUT: -

Registers affected after return: 

     A.BCDE../IXIY same
     .F....HL/.... different

Notes:

Internal operating system usage.
Clears buffer and sets buffer pointer.
Buffer length is 22 bytes.
