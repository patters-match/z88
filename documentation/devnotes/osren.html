OS_Ren, file rename

RST 20H, DEFW $E406

IN: 

     IX = DOR handle
     HL = pointer to new filename (null-terminated)
OUT, if call successfull: 
     Fc = 0
     IX = 0, DOR handle freed
OUT, if call failed: 
     Fc = 1
     IX = 0, DOR handle freed
Registers changed after return: 
     not documented.
Notes:
Internal operating system usage.
Applications must use GN_Ren.
