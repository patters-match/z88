GN_Soe, write string at extended address to standard output

RST 20H, DEFW $3C09

IN: 

     BHL = pointer to null-terminated string at extended address
     (B = 0, is interpreted as bank 0, not local...)
OUT, if call successful: 
     Fc = 0
     BHL points to null terminator
Registers changed after return: 
     AFBCDE../IXIY same
     ......HL/.... different
Notes:
If B is zero on entry the address will not be treated as local.
