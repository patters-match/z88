GN_D24, 24bit unsigned division

RST 20H, DEFW $7809

IN: 

     BHL = dividend
     CDE = divisor
OUT, if call successful: 
     Fc = 0
     BHL = quotient
     CDE = remainder
OUT, if call failed: 
     Fc = 1
     A = error code:
          RC_FAIL ($16), division by zero attempted)
Registers changed after return: 
     ......../IXIY same
     AFBCDEHL/.... different
