GN_Wcl, close wildcard handle

RST 20H, DEFW $5409

IN: 

     IX = wildcard handle
OUT, if call successful: 
     Fc = 0
     IX = 0
OUT, if call failed: 
     Fc = 1
     A = error code:
          RC_HAND ($08), bad handle
Registers changed after return: 
     ...C..HL/..IY same
     AFB.DE../IX.. different
