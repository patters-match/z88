OS_Use, fetch information about process card usage

RST 20H, DEFW $EE06

IN: 

     BHL = pointer to stacked process environment
     IX = static process id (handle)
     C = dynamic process id
OUT: 
     A0 = 1, if card slot 0 used
     A1 = 1, if card      1
     A2 = 1,              2
     A3 = 1,              3
Notes:
Internal operating system usage.

     -
