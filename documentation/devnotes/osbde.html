OS_Bde, copy bytes to extended address

RST 20H, DEFW $DA06

IN: 

     C = number of bytes to copy (0 ... 255)
     HL = local source pointer
     BDE = extended destination pointer
OUT: 
     Fc = 0 always
Registers changed after return: 
     A.BCDEHL/IXIY same
     .F....../.... different
