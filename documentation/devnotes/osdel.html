OS_Del, file delete

RST 20H, DEFW $E606

IN: 

     IX = DOR handle
OUT, if call successful: 
     Fc = 0
     IX = 0, DOR handle deleted.
OUT, if call failed: 
     Fc = 1
     IX = 0
          A = RC_???
Registers changed after return: 
     not documented.
Notes:
Internal operating system usage. Applications must use GN_Del.
