DC_Bye, exit current process

RST 20H, DEFW $080C

IN:  

A = return code 
     Use RC_OK for a standard exit

OUT: -

Notes: 

Internal operating system usage.
Does not return to caller.
Applications use OS_Bye to kill itself.

Bugs: -
