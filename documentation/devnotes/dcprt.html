DC_Prt, print to CLI

RST 20H, DEFW $120C

IN:

  A = character to output

OUT: -

Registers affected after return: 

     ..BCDEHL/IXIY same
     AF....../.... different

Notes:

Internal operating system usage.
Writes char to both prtstream and prtstreamT - close output streams if write error.
