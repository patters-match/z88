DC_Xin, examine CLI input

RST 20H, DEFW $1E0C

IN: -

OUT:

    Fc = 0 : CLI has input stream
  Fc = 1, A = RC_Eof : there is no CLI or no input stream

Registers affected after return: 

     ..BCDEHL/IXIY same
     AF....../.... different
