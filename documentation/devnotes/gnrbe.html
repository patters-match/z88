GN_Rbe, read byte at extended address

RST 20H, DEFW $3E09

IN: 

     BHL = pointer to the byte to read (B = 0, local byte)
OUT, if call successful: 
     A = byte at address
Registers changed after return: 
     .FBCDEHL/IXIY same
     A......./.... different
