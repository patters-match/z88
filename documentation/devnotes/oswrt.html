OS_Wrt, write token

RST 20H, DEFW $CC06

IN: 

     A = token (A7 = 1 always)
OUT: 
     Fc = 0
Registers changed after return: 
     A.BCDEHL/IXIY same
     .F....../.... different
Notes:
This will write the "menu topic help" token if a local token table has not been established with OS_Wtb.
