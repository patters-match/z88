OZ_Buf, buffer service routine 

CALL $0051

IN:

            L = reason call
OUT:
            Depends on the reason.

Registers changed after return:
            ..BCDEHL/IXIY same
            AF....../.... different

Notes:

Internal operating system useage. Removed since 0Z V4.1.
Circulary buffer structure have been fixed to 256 bytes to improve speed. All the calls to this routine are direct and all the code have been relocated in low ram for speed.
