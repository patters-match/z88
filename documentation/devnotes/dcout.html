DC_Out, write to CLI

RST 20H, DEFW $100C

IN:

  A = character to output

OUT: -

Registers affected after return: 

     AFBCDEHL/IXIY same
     ......../.... different

Notes:

Internal operating system usage.
Writes char to both outstream and outstreamT - close output streams if write error.
If there is no CLI, outputs to screen.
