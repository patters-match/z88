OS_Dly, delay a given period

RST 20H, DEFW $D606

IN: 

     BC = delay in centiseconds
OUT, if call failed: 
     Fc = 1
     BC = remaining timeout
     A = error code:
          RC_ESC ($01), escape was pressed
          RC_SUSP ($69), process suspended
          RC_TIME ($02), timeout
Registers changed after return: 
     ....DEHL/IXIY same
     AFBC..../.... different
