OS_Pout, write embedded null-terminated string at caller (PC) to standard output.

RST 20H, DEFB $93

IN: 

     -

OUT: 
   Fc = 0 always (error handler is never provoked)

Registers changed after return: 
   A.BCDEHL/IXIY same
  .F....../.... different

Example: 
  ...
  OZ  OS_Pout
  defm 1,"7#4",$20+1,$20+0,$20+18,$20+8,$83
  defm 1,"2C4"
  defm 1,"2JC", 0
  ...
  ret
