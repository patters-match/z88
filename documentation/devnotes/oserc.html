OS_Erc, get error context

RST 20H, DEFB $72

IN: 

     -
OUT: 
     A = last error code
     B = resumption cycle
     C = dynamic id
     IX = reserved
     Fc = 0, always
Registers changed after return: 
     ....DEHL/..IY same
     AFBC..../IX.. different
Notes:
Internal operating system usage.
BC, IX are reserved for future use.
