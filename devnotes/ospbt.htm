OS_Pbt, write byte to file, device with timeout

RST 20H, DEFB $42

IN: 

     A = byte to be written to file, device
     BC = timeout in 1/100 seconds
     IX = handle of file, device
OUT, if call successful: 
     Fc = 0
     BC = remaining in 1/100 seconds
OUT, if call failed: 
     Fc = 1
     A = error code:
          RC_HAND ($08), bad handle
          RC_WP ($14), write protected
          RC_ESC ($01), escape, if device is :COM.0, :PRT.0
          RC_TIME ($02), timeout
Registers changed after return: 
     ....DEHL/IXIY same
     AFBC..../.... different
