GN_Pmd, put (set) the current machine date

RST 20H, DEFW $1C09

IN: 

     HL = 0, not allowed

     HL = 1, not allowed

     HL = 2, 3 byte internal date in ABC (A = high byte, C = low byte)

     HL > 255, HL points to 3 byte internal date (low byte first)
OUT, if call successful: 
     -
Registers changed after return: 
     AFBCDEHL/IXIY same

     ......../.... different
Notes:
The date is used throughout the machine and should only be altered if the user of the machine is fully aware of the fact.

Flow sequense:
1. Get real time in days/seconds/centiseconds
2. Reset RTC
3. Multiply set-days by $011580
4. Add real seconds.
5. Save new base seconds
